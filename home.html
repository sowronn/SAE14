Dans le cadre de la SAE 14, voilà le site internet demander. Son but : Créer deux page web. L’une sur notre hobby et l'autre pour nous présenter, la page lier a notre hobby doit être traduite en anglais du moins en partie^^.
Cette phrase na pas véritablement de but elle est principalement la pour, premièrement rappeler le but de ces pages ainsi qu’ aider à la compréhension du projet. deuxièmement, pour ... remplir un peu cette page home qui ma fois est un peu triste vide je le rappelle.


Mais heureusement qu’il y a la possibilité de rajouté tout ce texte me permet de rappeler l'intérêt de ce projet. Ce dernier étant de nous familiariser avec les technologie web tel que l'HTML et le CSS, de montré nos compétence en web, en design. Pour moi ces page internet est aussi dédier a ce que les profs qui lise le contenue connaisse mieux les élèves, si c’est le cas c’est très bien joué. :)


Vous l'aurez compris cette page se doit d'être la plus simple possible dans le but de laisser du contenue pour les autres pages.
